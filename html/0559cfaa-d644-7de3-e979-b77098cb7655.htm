EntityRepository Class

Mixture of both the Repository and Singleton pattern Entity repository acts as a central storage for all current entities within the game, ensuring that only one copy of each entitiy exists and all entites are updated/rendered only once per game loop. Can be accessed by other classes where needed. Within a model-view-controller pattern this class acts as the model database
